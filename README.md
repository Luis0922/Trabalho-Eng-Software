# Para rodar o projeto
- Clonar o repositorio
- No terminal:
  - php artisan key:generate
  - php artisan migrate
  - php artisan db:seed
- Dividir o terminal:
  - Terminal 1: php artisan serve
  - Terminal 2: npm run dev
- Abrir no navegador:
  - http://127.0.0.1:8000
